# CS Data Structures & Algorithms

This repository contains my from-scratch implementations of core data structures and algorithms as part of my Computer Science fundamentals preparation.

## Purpose
The goal of this project is to demonstrate understanding of core Computer Science concepts including data organization, algorithmic efficiency, and problem-solving.

## Implemented Data Structures
- Arrays
- Linked Lists
- Stack
- Queue
- Binary Search Tree
- Graph (Adjacency List)

## Implemented Algorithms
- Searching (Linear, Binary)
- Sorting (Bubble, Merge, Quick)
- Graph Traversal (BFS, DFS)

## Technologies
- Python 3
